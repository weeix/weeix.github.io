วันหนึ่งก็เกิดสงสัย ว่าบริการจดโดเมนปีละ 340.- เนี่ย มันต่างจากแบบปีละ 250.- ยังไง  ก็เลยลอง transfer ไปใช้แบบหลังดู ปรากฏว่า..

1.  ล็อค-ปลดล็อกโดเมนเองไม่ได้
2.  ดู domain secret ไม่ได้
3.  ไม่มีระบบอีเมล
4.  ซ่อนข้อมูลใน WHOIS (privacy protection) ไม่ได้
5.  ไม่มีเครื่องมือจัดการ DNS record

เจ้าข้อสุดท้ายนี้เองทำให้ต้องไปตามหาว่ามีที่ไหนให้บริการ Authoritative DNS Server บ้าง

เลยมีโอกาสได้รู้จักกับ [Cloudflare](https://www.cloudflare.com) ซึ่งก่อนหน้านี้ก็เคยได้ยินมาเหมือนกันว่าเขาเปิดให้บริการ CDN ฟรี แต่ทว่าตอนนั้นแทบไม่มี traffic วิ่งเข้าเว็บเลย จึงไม่ค่อยสนใจสักเท่าไหร่  จะว่าไปตอนนี้ก็คงไม่ค่อยแตกต่างจากตอนนั้นหรอก 555  แต่ที่พูดขึ้นมาเนี่ย เป็นเพราะไปค้นเจอว่าเขาให้บริการเกี่ยวกับ DNS ด้วย

อันที่จริงแล้วมันก็เป็นส่วนหนึ่งของบริการเขานั่นแหละ คือถ้าใครอยากใช้บริการ CDN ฟรีของ Cloudflare ก็ต้องไปตั้งค่าโดเมนตัวเองให้ชี้ไปหา DNS Server ของ Cloudflare ก่อน

เรียกได้ว่ายิงปืนนัดเดียวได้นกสองตัว ..ไม่สิ ได้นกหลายตัวเลย เพราะนอกจากจะได้ใช้บริการ DNS ที่มีหน้าเว็บไว้จัดการ record แบบง่ายๆ กับ CDN ที่มีปุ่มเปิด/ปิดอยู่ท้ายแต่ละ record แล้ว ยังมีบริการอื่นๆติดมาอีกเพียบ หนึ่งในนั้นก็คือบริการเข้ารหัสหน้าเว็บของเราด้วย SSL ที่แทบจะไม่ต้องตั้งค่าอะไรเพิ่มเติมเลย (ไว้คราวหน้าจะเล่าให้ฟัง) ซึ่งทั้ง 3 อย่างที่กล่าวมานั้น (DNS, CDN และ SSL) มาพร้อมกับแพคเก็จที่ Cloudflare เปิดให้เราใช้ฟรี

"เย้"

ที่เหลือก็แค่ไปเพิ่ม record ใน DNS ตามที่ผู้ให้บริการแต่ละเจ้าบอกเท่านั้นเอง (ไว้คราวหน้าของคราวหน้าจะมาเล่าให้ฟัง)
