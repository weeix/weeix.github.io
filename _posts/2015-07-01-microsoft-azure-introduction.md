ถ้าพูดถึง Microsoft หลายๆคนคงรู้จักดี :) บริษัทที่ผลิตระบบปฏิบัติการ Windows มาให้เราใช้นั่นเอง เดี๋ยวนี้ Microsoft หันมาให้บริการทั้ง Iaas และ Paas แล้วนะ (ตั้งแต่ต้นปี 2553)

ถ้าใครอยากใช้ของฟรี เอ้ย อยากทดสอบการใช้งานก่อน สามารถสมัครใช้ [Free Trial subscription](https://azure.microsoft.com/en-us/pricing/free-trial) ได้ ตอนสมัครต้องใช้ข้อมูลบัตรเครดิตนะ เขาจะให้เราทดลองใช้ฟรี 1 เดือน โดยมีวงเงินให้ $200 หรือคิดเป็นเงินไทยตอนนี้ก็ประมาณ 6,800฿ แล้วแต่ว่าอันไหนจะหมดก่อนกัน พอเวลาหรือวงเงินที่ได้รับมาหมดแล้ว บริการของเราก็จะโดนระงับไป ถ้าเกิดติดใจอยากใช้บริการต่อก็สามารถอัพเกรดจาก Free Trial subscription เป็น Pay-As-You-Go Azure subscription ที่แปลว่าใช้เท่าไหร่จ่ายเท่านั้น หรือถ้าเราไม่คิดจะใช้ต่อก็ปล่อยไว้อย่างนั้น ทาง Microsoft จะไม่ตัดเงินจากบัตรเครดิตของเรา อันนี้อ้างอิงจาก [Azure Free Trial FAQ](//azure.microsoft.com/en-us/pricing/free-trial-faq/) แต่..

## ไม่อยากเสียตัง อ่านตรงนี้ก่อน

$200 ที่เราได้มาเนี่ย ไม่เกี่ยวกับบริการต่างๆใน Marketplace นา ถ้าเราเผลอไปกดใช้เข้า ก็อาจจะเสียเงินได้ในบัดดล อันนี้ทีม support จาก Microsoft บอกมา พร้อมทั้งส่งลิงค์ [Marketplace FAQs](//azure.microsoft.com/en-us/marketplace/faq/) มาให้อ่าน ก่อนจะใช้บริการอะไรก็ดูก่อนนะ ว่ามาจาก Microsoft รึเปล่า ถ้าไม่ใช่ก็ดูอีกทีว่ามีค่าบริการไหม คิดค่าบริการยังไง จ่ายตามที่ใช้จริงหรือจ่ายเป็นรายเดือน/รายปี คืออันที่ฟรีก็มีนะ บอกเอาไว้ก่อน เดี๋ยวจะระแวงจนไม่กล้าเปิด Marketplace 55

ในหน้าจัดการแบบใหม่ (ที่จะกล่าวถึงในหัวข้อถัดไป) บริการจาก Marketplace บางอันก็ไม่ได้อยู่ในโซน Marketplace นะ  อย่างก่อนหน้านี้ ผมเคยพยายามจะติดตั้ง Web App ที่ชื่อว่า Scalable Wordpress ซึ่งอยู่ในเมนู NEW > Web App เสมือนว่าเป็นบริการที่ Microsoft จัดหามาให้ โดย Wordpress เนี่ย รู้ๆกันอยู่ว่ามันเป็นซอฟต์แวร์โอเพนซอร์ส และสามารถหาดาวน์โหลดได้ฟรี ส่วนฐานข้อมูลที่ใช้ก็จะเป็นของ ClearDB ซึ่งมีค่าใช้จ่ายเริ่มเดือนละประมาณ $3.5 (มารู้ทีหลังว่าเขาคิดเป็นรายเดือน ไม่ได้จ่ายตามที่ใช้จริงอย่างที่เข้าใจไว้) โดย**ค่าบริการ ClearDB จะไม่หักจาก Free Trial credit $200 ที่เราได้รับ แต่จะหักจากบัตรเครดิตของเราแทน**

ตอนนั้นก็พยายามติดตั้งอยู่พักนึง แต่ระบบแจ้งว่าติดตั้งไม่สำเร็จ เลยไปลบซากที่มากับการติดตั้ง เช่น ฐานข้อมูล เครือข่ายจำลอง ฯลฯ ทิ้ง แล้วพยายามติดตั้งใหม่ ผลก็เหมือนเดิม เลยเลิกล้มความตั้งใจเสีย  มารู้ทีหลังจากการสอบถามทีม support ของ MS Azure ว่าเวลาติดตั้งเนี่ย ต้องเลือกฐานข้อมูลเป็นแพ็คเกจ Saturn (เดือนละ $50) ขึ้นไป แหมะ แล้วก็ไม่เขียนบอกไว้ตั้งแต่แรก ใครมันจะไปรู้ล่ะครับพี่น้อง

แต่ดีแล้วล่ะที่ไม่เลือกแพ็คเกจ Saturn ในตอนนั้น เพราะอีกเดี๋ยวเดียวก็มีบิลส่งเข้าเมล 2 ฉบับ แจ้งยอดค่าใช้จ่ายเป็นสกุลเงินไทยเรียบร้อย ฉบับละ 110฿ ซึ่งก็รู้ได้ทันทีว่ามาจากยอด $3.5 ของ ClearDB ที่ติดมากับ Scalable Wordpress แหงมๆ ทำให้ต้องรีบ login กลับเข้าไปลบซาก ClearDB อันที่สองในบัดดล พอลบได้สักพักก็มีเมลอีกฉบับแจ้งมาว่ายกเลิกบริการแล้วนะ แต่ทว่าเรื่องยังไม่จบเพียงเท่านั้น

พอบิลค่าบัตรเครดิตมา อ้าว ยังมีรายการอยู่นี่นา เลยไปปรึกษากับทีม support คนนึงที่เราเคยถามเขาว่าค่า ClearDB นี่หักเงินจากวงเงิน Free Trial credit $200 ไหม คำตอบที่ได้รับคราวนั้นคือ ใช่.. (คิดดูขนาดทีม support ของ Microsoft ยังเข้าใจผิดอ่ะ แล้วคนนอกอย่างเราๆจะรอดมั้ยเนี่ย 55) เขาก็เลยขอให้เปิดเคส refund แล้วแจ้งหมายเลขเคสให้เขา เขาจะเดินเรื่องให้

สุดท้ายทุกอย่างก็เรียบร้อยดี ได้รับเงินคืนในรอบบัตรเครดิตเดือนถัดไป

## หน้าจัดการมีทั้งแบบเก่า และแบบใหม่

กลับมาเรื่องของการใช้งานดีกว่า ถ้าใครกำลังใช้ Management Portal ของ Azure ในช่วงนี้ จะพบว่าจะมีให้เราเลือกใช้ 2 แบบ ได้แก่

1.  แบบเก่า - [https://manage.windowsazure.com](https://manage.windowsazure.com)
2.  แบบใหม่ - [https://portal.azure.com](https://portal.azure.com)

[![หน้าจัดการแบบเก่า](//weeix.com/wp-content/uploads/2015/06/017-classic-home-1024x600.png)](//weeix.com/wp-content/uploads/2015/06/017-classic-home.png)

[![หน้าจัดการแบบใหม่](//weeix.com/wp-content/uploads/2015/06/003-dashboard.png)](//weeix.com/wp-content/uploads/2015/06/003-dashboard.png)

ซึ่งแบบใหม่นี้ยังอยู่ในช่วง preview ซึ่งก็คือช่วงเปิดให้ทดสอบนั่นเอง หมายความว่ามันจะยังทำงานเอ๋อๆ และมีข้อบกพร่องอยู่บ้าง ในส่วนข้อมูลก็จะมีถูกบ้างผิดบ้าง ถ้าใครไม่ค่อยมั่นใจ แนะนำให้ใช้แบบเก่าไปก่อน เดี๋ยวกดพลาดไปซื้อของใน Marketplace แล้วจะเสียตังค์เอา (พูดในฐานะที่เคยพลาดมาแล้ว 55)

แต่งานบางอย่างก็ทำได้เฉพาะในหน้าจัดการแบบเก่า บางอย่างก็ทำได้เฉพาะในหน้าจัดการแบบใหม่ ตัวอย่างเช่น ถ้าจะจอง IP (ทำให้ได้ IP เดิมเวลาปิดเครื่องแล้วเปิดใหม่) ให้เครื่อง VM ตอนสร้างมันขึ้นมาใหม่เนี่ย ต้องทำในหน้าจัดการแบบใหม่นะ แบบเก่ามันไม่มีให้เลือก

เพราะฉะนั้นถ้าเราเป็นมือใหม่ก็ใช้หน้าจัดการแบบเดิมไปก่อน พอคล่องและมั่นใจแล้วค่อยไปลองหน้าจัดการแบบใหม่

ไว้เจอกันคราวหน้า ถ้าเจออะไรน่าสนใจเดี๋ยวจะมาเล่าให้ฟัง
